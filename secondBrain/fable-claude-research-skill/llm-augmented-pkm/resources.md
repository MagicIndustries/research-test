# Resources — LLM-Augmented Classic PKM

Star counts checked via the GitHub API on 2026-07-18.

## Methodology

- [Building a Second Brain — official site](https://www.buildingasecondbrain.com/) — Tiago Forte's book, courses, and free resources; origin of PARA and CODE.
- [The AI Second Brain (Forte Labs)](https://www.buildingasecondbrain.com/ai-second-brain) — Forte's first-party take on LLMs + BASB: "personal context management," the Master Prompt, PARA adapted for AI; live cohort course (next: Sep–Oct 2026).
- [PARA Method explained (Forte Labs blog)](https://fortelabs.com/blog/para/) — the canonical free write-up of PARA.
- [Introduction to the Zettelkasten Method (zettelkasten.de)](https://zettelkasten.de/introduction/) — the standard modern reference for Luhmann's method.
- [Personal Knowledge Management (2026): The Practical Guide](https://www.atlasworkspace.ai/blog/personal-knowledge-management) — good synthesis of how PKM practice changed once notes acquired an LLM audience.

## Tools

- [Obsidian](https://obsidian.md) — the vault app. Free personal use; [Sync](https://obsidian.md/sync) and Publish are paid. Data = local Markdown.
- [Obsidian Web Clipper](https://obsidian.md/clipper) — official browser extension; article → clean Markdown in your vault.
- [Logseq](https://github.com/logseq/logseq) — 43.9k★, AGPL-3.0, active. Open-source outliner alternative; block references, daily-notes-first.
- [Ollama](https://ollama.com) — local model runner; pairs with the plugins below for a fully offline AI vault.

## Obsidian AI plugins

- [Copilot for Obsidian](https://github.com/logancyang/obsidian-copilot) — 7.4k★, AGPL-3.0, pushed this week. Chat sidebar, vault QA, semantic search, inline edits; cloud or local models. [Docs](https://www.obsidiancopilot.com/).
- [Smart Connections](https://github.com/brianpetro/obsidian-smart-connections) — 5.3k★, active. Passive related-notes pane + local embeddings, zero setup. [Site](https://smartconnections.app).
- [Text Generator](https://github.com/nhaouari/obsidian-textgenerator-plugin) — 2.0k★, MIT, updated May 2026. Prompt-template text generation with any provider.
- [Plugin directory: AI category](https://obsidian.md/plugins?search=ai) — browse what's current; the ecosystem moves fast.
