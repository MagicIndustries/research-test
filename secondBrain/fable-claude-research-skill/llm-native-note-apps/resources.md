# Resources — LLM-Native Note Apps

## Official products & docs

- [Notion](https://www.notion.com) · [Notion AI](https://www.notion.com/product/ai) — workspace + AI; see the [AI data-usage FAQ](https://www.notion.com/help/notion-ai-security-practices) before storing sensitive material.
- [Tana](https://tana.inc) · [Tana docs on supertags](https://tana.inc/docs) — the node/supertag model that defines the product.
- [Mem](https://mem.ai) — auto-organizing AI notes; ~$12/mo.
- [Reflect](https://reflect.app) · [Reflect docs](https://reflect.app/docs) — E2EE networked notes, $10/mo, multi-model AI (GPT-4o/Claude/Gemini).
- [Capacities](https://capacities.io) — object-based notes; generous free tier, Pro ~$11.99/mo.
- [NotebookLM](https://notebooklm.google) — Google's source-grounded research tool; free tier, Pro via Google One AI. [Official help](https://support.google.com/notebooklm/).
- [Heptabase](https://heptabase.com) — visual/whiteboard-first knowledge tool, frequently on 2026 second-brain lists ([Buildin roundup](https://buildin.ai/blog/best-second-brain-apps-2026)).

## Caveats worth reading before subscribing

- All of these are closed-source SaaS; check current AI-training/data-retention policies per vendor (they change).
- Export formats: Notion → Markdown/CSV (loses databases' relations), Tana → JSON (supertag semantics don't transfer), Mem/Reflect → Markdown. Test the exit early.
