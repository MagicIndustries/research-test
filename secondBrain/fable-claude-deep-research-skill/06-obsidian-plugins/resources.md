## Trial instructions (~30 min total)

1. Obsidian → Settings → Community plugins → install **Smart Connections** and **Copilot**.
2. Let Smart Connections index the vault (bundled local embeddings, no key needed); open the connections pane on a note you know well and judge the suggestions.
3. Configure Copilot with either an Anthropic/OpenAI key or local Ollama (`http://localhost:11434/v1`); run a Vault QA question that spans multiple notes.
4. Optional: add the Khoj plugin later if you adopt Khoj server-side.
