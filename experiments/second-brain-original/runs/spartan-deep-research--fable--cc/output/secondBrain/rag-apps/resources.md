# Self-Hosted RAG Apps — Resources

All links accessed 2026-07-18; star counts verified that day where noted.

## GitHub repos

- **Mintplex-Labs/anything-llm** — ~63.5k stars (verified), MIT. All-in-one private ChatGPT: desktop + Docker, agents, MCP, 25+ LLM providers, LanceDB/PGVector/Pinecone/Chroma/Weaviate/Qdrant/Milvus. https://github.com/Mintplex-Labs/anything-llm ; site: https://anythingllm.com
- **khoj-ai/khoj** — ~35.8k stars (verified), AGPL-3.0. Self-hostable AI second brain; Obsidian/Emacs/browser/desktop/phone/WhatsApp clients; custom agents, automations, deep research; local (Llama, Qwen, Mistral…) or hosted (GPT, Claude, Gemini, DeepSeek) models; Docker Compose self-hosting. Latest release 2.0.0-beta.28 (2026-03-26). https://github.com/khoj-ai/khoj ; docs: https://docs.khoj.dev ; cloud: https://app.khoj.dev
- **QuivrHQ/quivr** — "second brain" RAG platform, brain-based organization, Supabase backend. https://github.com/QuivrHQ/quivr
- **reorproject/reor** — ~8.6k stars, AGPL-3.0, **archived 2026-03-07** (read-only). Local AI PKM: auto-linked notes, semantic search, local Q&A; Ollama/Transformers.js/LanceDB stack. Historically important; data is portable markdown. https://github.com/reorproject/reor ; site: https://reorproject.org
- **neuml/txtai** — embeddings database / RAG framework if you'd rather build your own pipeline in Python. https://github.com/neuml/txtai
