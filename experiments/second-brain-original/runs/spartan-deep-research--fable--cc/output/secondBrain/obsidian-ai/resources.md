# Obsidian + AI — Resources

All links accessed 2026-07-18.

## Plugins (official listings)

- **Smart Connections** — local-embedding semantic linking. https://community.obsidian.md/plugins/smart-connections
- **Copilot for Obsidian** — chat with vault; supports Ollama/LM Studio/OpenAI-compatible endpoints. https://community.obsidian.md/plugins/copilot
- **Simple Graph Builder** — LLM-powered entity extraction building a lightweight knowledge graph from your notes (Claude, OpenAI, Gemini, Ollama). Bridges to the graph method. https://community.obsidian.md/plugins/simple-graph-builder ; repo: https://github.com/junhewk/simple-graph-builder

## MCP servers & agent bridges (GitHub)

- **coddingtonbear/obsidian-local-rest-api** — the foundational community plugin: secure REST API for the vault; now ships a built-in MCP server. https://github.com/coddingtonbear/obsidian-local-rest-api
- **MarkusPfundstein/mcp-obsidian** — the most-referenced MCP server; tools: `list_files_in_vault`, `get_file_contents`, `search`, `patch_content`, `append_content`, `delete_file`. Requires the Local REST API plugin + API key. https://github.com/MarkusPfundstein/mcp-obsidian
- **iansinnott/obsidian-claude-code-mcp** — connect Claude Code and other MCP clients to the vault from inside Obsidian. https://github.com/iansinnott/obsidian-claude-code-mcp
- **PublikPrinciple/obsidian-mcp-rest** — Node.js MCP implementation for Claude Desktop. https://github.com/PublikPrinciple/obsidian-mcp-rest
- **kanishkez/obsidian-mcp** — minimal MCP server for Claude Code direct vault editing. https://github.com/kanishkez/obsidian-mcp
- **eugeniughelbur/obsidian-second-brain** — full agent-maintained-vault skill (see [../llm-wiki/resources.md](../llm-wiki/resources.md)). https://github.com/eugeniughelbur/obsidian-second-brain
- Directory of Obsidian MCP servers: https://mcpservers.org/servers/MarkusPfundstein/mcp-obsidian
