# Agent Memory — Resources

*Collected 2026-07-18.*

## GitHub repositories

- **mem0ai/mem0** — the most-adopted agent memory library (~41k stars). https://github.com/mem0ai/mem0
- **getzep/graphiti** — temporal knowledge-graph memory engine (Zep); Apache-2.0. https://github.com/getzep/graphiti
- **letta-ai/letta** — Letta (formerly MemGPT): memory-OS agents with self-editing memory blocks. https://github.com/letta-ai/letta
- **basicmachines-co/basic-memory** — MCP server: Claude reads/writes a local markdown knowledge base; Obsidian-compatible. https://github.com/basicmachines-co/basic-memory
- **TeleAI-UAGI/Awesome-Agent-Memory** — curated list of memory systems, benchmarks, and papers. https://github.com/TeleAI-UAGI/Awesome-Agent-Memory
