# Wikis & Digital Gardens — Resources

*Collected 2026-07-18.*

## GitHub repositories

- **jackyzha0/quartz** — the leading digital-garden static-site generator; MIT. https://github.com/jackyzha0/quartz
- **MaggieAppleton/digital-gardeners** — the community hub: resources, tools, example gardens, ethos. https://github.com/MaggieAppleton/digital-gardeners
- **kevin7lou/wiki** — working template: Obsidian + Quartz + GitHub Actions personal wiki. https://github.com/kevin7lou/wiki
- **requarks/wiki** — Wiki.js, modern Node.js wiki engine. https://github.com/requarks/wiki
- **TiddlyWiki/TiddlyWiki5** — single-file non-linear wiki. https://github.com/TiddlyWiki/TiddlyWiki5
- **oleeskild/obsidian-digital-garden** — free Obsidian→Netlify publishing plugin. https://github.com/oleeskild/obsidian-digital-garden
