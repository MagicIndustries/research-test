# Resources: Open-Source Research Frameworks

## Primary — repositories

- [assafelovic/gpt-researcher](https://github.com/assafelovic/gpt-researcher) — 28.4k★, Apache-2.0, active (v3.5.1, Jun 2026). Planner/executor agent; deep (recursive) mode ~$0.40/report per README; MCP; LangGraph multi-agent report team. Site: [gptr.dev](https://gptr.dev/).
- [stanford-oval/storm](https://github.com/stanford-oval/storm) — 30.1k★, MIT. STORM + Co-STORM reference implementations; four-stage pipeline; slowed since v1.1.0 (Jan 2025).
- [huggingface/smolagents — open_deep_research example](https://github.com/huggingface/smolagents/tree/main/examples/open_deep_research) — CodeAgent-based reproduction; GAIA 55.15% validation.
- [dzhng/deep-research](https://github.com/dzhng/deep-research) — 19.4k★, MIT, TypeScript, <500 lines; breadth/depth recursive loop.
- [langchain-ai/open_deep_research](https://github.com/langchain-ai/open_deep_research) — supervisor/researchers on LangGraph (see multi-agent-pipelines/resources.md).

## Primary — papers

- [Assisting in Writing Wikipedia-like Articles From Scratch with LLMs — STORM (arXiv:2402.14207, NAACL 2024)](https://arxiv.org/abs/2402.14207) — perspective-guided question asking; +25 pts organization, +10 pts breadth in expert judgment.
- [Co-STORM (arXiv:2408.15232, EMNLP 2024)](https://arxiv.org/abs/2408.15232) — collaborative human-AI discourse extension.
- [Improving and Evaluating Open Deep Research Agents (arXiv:2508.10152)](https://arxiv.org/pdf/2508.10152) — academic follow-up on strengthening open agents.
- [DeepScholar-Bench (arXiv:2508.20033)](https://arxiv.org/pdf/2508.20033) — live benchmark for generative research synthesis; useful for testing open frameworks without contamination.
