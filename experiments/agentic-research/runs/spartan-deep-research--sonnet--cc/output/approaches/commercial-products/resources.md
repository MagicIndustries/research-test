# Resources: Commercial Deep-Research Products

Annotated sources used for the commercial-products section, grouped by type. Every performance/superiority claim sourced from a vendor is flagged "(vendor self-report)."

- **Gemini Deep Research Agent — Gemini API docs** — https://ai.google.dev/gemini-api/docs/interactions/deep-research — dated Jul 14, 2026 (preview). - **Introducing Perplexity Labs** — https://www.perplexity.ai/hub/blog/introducing-perplexity-labs — May 29, 2025. Launch of the broader agentic workspace (reports/spreadsheets/dashboards/web apps) built on the same research loop. [primary]
- **Claude takes research to new places** — https://www.anthropic.com/news/research — Apr 2025. Anthropic's Research feature launch: multi-step web + Google Workspace search, early beta on Max/Team/Enterprise in US/Japan/Brazil. Claims Claude answers "in minutes" vs. rivals' "up to 30 minutes." [primary] (vendor self-report / comparative marketing claim, not independently verified)
- **Deep Research in Microsoft Copilot — Microsoft Support** — https://support.microsoft.com/en-us/microsoft-copilot/deep-research-in-microsoft-copilot — current, notes consumer Copilot "Deep Research" is being retired Aug 18, 2026; "Researcher" (which wraps an OpenAI deep-research model with Microsoft orchestration) continues for Microsoft 365 Premium subscribers. - **You.com Research API announcement** — https://you.com/resources/research-api-by-you-com — Feb 2026. Claims #1 on DeepSearchQA (83.67% accuracy, 93.16% F1). [primary] (vendor self-report)

## Repos

- **openai/openai-python** — https://github.com/openai/openai-python — actively maintained (latest release v2.46.0, Jul 17, 2026, confirmed live). Official Python SDK for the OpenAI API, including the Responses API used to call Deep Research-capable models programmatically. Not Deep-Research-specific. [primary]
- **ppl-ai/api-cookbook** — https://github.com/ppl-ai/api-cookbook — confirmed live, current. Perplexity's official collection of Sonar API integration examples (fact-checking, financial/academic-research use cases); does not currently include a dedicated `sonar-deep-research` example. [primary]
- Note: no dedicated first-party "Deep Research SDK" repo was found for Google; Gemini Deep Research is accessed through the general Gemini API / `google-genai` client and the newer Interactions API described in vendor docs above — a standalone repo reference was not independently verified in this session (web-search budget was exhausted before this could be confirmed).

## Papers

- **DeepResearch Bench II: Diagnosing Deep Research Agents via Rubrics from Expert Report** — arXiv:2601.08536 — https://arxiv.org/abs/2601.08536 — Feb 2, 2026. The most methodologically credible independent comparison found: rubric-based scoring (Recall/Analysis/Presentation) derived from human-expert reports, evaluated against OpenAI-GPT-o3 Deep Research, Gemini-3-Pro/Gemini-2.5-Pro Deep Research, Perplexity Research, Grok Deep Search, and others, snapshotted ~November 2025 (predates Feb/Apr 2026 vendor rebuilds). [secondary/independent]
- **DeepResearch Bench II — GitHub companion** — https://github.com/imlrz/DeepResearch-Bench-II — companion code/data repo for the above paper. [secondary/independent]
- **A Comprehensive Survey of Deep Research: Systems, Methodologies, and Applications** — arXiv:2506.12594 — https://arxiv.org/pdf/2506.12594 — 2025 survey covering the broader deep-research agent landscape (background/context, not 2026-specific). [secondary]
- **OpenResearcher: A Fully Open Pipeline for Long-Horizon Deep Research Trajectory Synthesis** — arXiv:2603.20278 — https://arxiv.org/pdf/2603.20278 — 2026. Open-source alternative pipeline; useful contrast case for what "open" vs. "commercial closed" deep research looks like. [secondary]
