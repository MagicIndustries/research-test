# Commercial Deep-Research Products

## What counts as a "commercial deep-research product"

A commercial deep-research product is a hosted, vendor-operated agent feature — accessed through a chat UI and/or a metered API — that takes a natural-language research question, autonomously plans and executes a multi-step web (and sometimes private-data) search-read-reason loop over several minutes, and returns a long-form, citation-backed report without the user orchestrating individual tool calls. The category is defined by three properties that distinguish it from open-source research-agent frameworks: it is **closed-weight and closed-pipeline** (the underlying model, the planning/orchestration logic, and the ranking of retrieved sources are not inspectable or modifiable); it is **subscription- or usage-gated** (paid tiers, per-task credits, or capped monthly query allowances rather than free self-hosting); and it is **black-box-to-semi-transparent** (vendors show intermediate search queries and a source list, and increasingly stream reasoning steps, but the actual chain-of-thought and retrieval-ranking heuristics remain proprietary). The three flagship examples are OpenAI's Deep Research in ChatGPT, Google's Gemini Deep Research, and Perplexity's Deep Research/Labs; Anthropic, xAI, Microsoft and You.com run smaller but active competitors in the same category.

## How each product works

**OpenAI Deep Research** launched February 3, 2025, as a specialized post-trained variant of o3, with a "lightweight" o4-mini-based tier added in April 2025 for free/lower-tier users ([OpenAI launch post](https://openai.com/index/introducing-deep-research/), [Wikipedia summary](https://en.wikipedia.org/wiki/ChatGPT_Deep_Research)). By February 2026 the feature was rebuilt on a GPT-5.2-based model, adding scoped browsing (restrict search to trusted sites), MCP-server/app connectivity for private data, real-time progress streaming with mid-run interruption/refinement, and an improved report UI ([Neowin, Feb 2026](https://www.neowin.net/news/openai-upgrades-chatgpt-deep-research-with-gpt-52-and-real-time-controls/)). It autonomously browses the web for roughly 5–30 minutes, can read images and PDFs, and since July 2025 uses ChatGPT's visual browser as part of agent mode. Output is a single long cited report. OpenAI's own headline benchmark claim — 26.6% on Humanity's Last Exam at launch, versus 9.4% for DeepSeek R1 and 3.3% for GPT-4o — is a **vendor self-report** from the original 2025 announcement and has not been refreshed publicly for the GPT-5.2 version.

**Google Gemini Deep Research** launched December 11, 2024, for Gemini Advanced subscribers only, initially on Gemini 1.5 Pro, then upgraded through Gemini 2.0 Flash Thinking and made available to free-tier users with capped monthly runs ([TechCrunch, Dec 2024](https://techcrunch.com/2024/12/11/gemini-can-now-research-deeper/)). The most significant 2026 change: on April 21, 2026 Google shipped a developer-facing "Interactions API" with two async agents — `deep-research-preview-04-2026` (speed-optimized) and `deep-research-max-preview-04-2026` (thoroughness-optimized) — both rebuilt on Gemini 3.1 Pro with "significantly reduced latency and cost" ([Google blog, Apr 2026](https://blog.google/innovation-and-ai/models-and-research/gemini-models/next-generation-gemini-deep-research/); [Gemini API docs](https://ai.google.dev/gemini-api/docs/interactions/deep-research)). It runs a disclosed Plan→Search→Read→Iterate→Output loop using Google Search grounding (80 queries/task standard, 160/task Max), a URL-context reader, code execution, MCP tool connections, and file search over uploaded documents; tasks can run up to 60 minutes (most finish in under 20). Output includes cited reports, generated charts/infographics, and multimodal input support (PDF, CSV, image, audio, video). Google's claim of a "leap in performance across industry-standard benchmarks" versus its own December 2025 release is a vendor self-report with no third-party number cited alongside it.

**Perplexity Deep Research** launched February 14, 2025, free for all users with Pro subscribers getting higher daily limits ([TechCrunch, Feb 2025](https://techcrunch.com/2025/02/15/perplexity-launches-its-own-freemium-deep-research-product/)). It runs on the `sonar-deep-research` model (128K context, built on Perplexity's Sonar line) and performs iterative rounds of search, source-reading, and reasoning before synthesis — a launch-day API response example showed 21 search queries and ~194K reasoning tokens for one report ([Perplexity Sonar docs](https://docs.perplexity.ai/docs/sonar/models/sonar-deep-research)). Perplexity's launch-time HLE claim was 21.1%, ahead of Gemini Thinking (6.2%) but behind OpenAI's contemporaneous 26.6% — again a vendor self-report ([Perplexity launch blog](https://www.perplexity.ai/hub/blog/introducing-perplexity-deep-research)). On May 29, 2025, Perplexity added **Labs**, a broader agentic workspace (Pro-only) that extends the same search/reasoning loop with code execution and chart generation to produce spreadsheets, dashboards, and simple web apps in 10+ minutes of autonomous work ([TechCrunch](https://techcrunch.com/2025/05/29/perplexitys-new-tool-can-generate-spreadsheets-dashboards-and-more/)).

## Maturity, activity, and independent evidence

All three products are now 18+ months old and have shipped material 2026 revisions rather than sitting still: OpenAI's GPT-5.2 rebuild and MCP connectivity (Feb 2026), Google's Gemini 3.1-based dual-agent API (Apr 2026), and Perplexity's Labs expansion (2025, still active in 2026) with Pro/Max tier restructuring. OpenAI states 9M+ paying business users and 1M+ business customers as of November 2025, though this figure is company-wide, not Deep-Research-specific ([OpenAI enterprise report](https://openai.com/index/the-state-of-enterprise-ai-2025-report/)) — treat as a vendor self-report and only weak evidence of Deep Research adoption specifically, since no product-level usage numbers were found for any of the three vendors.

The most credible independent evaluation found is **DeepResearch Bench II** (arXiv 2601.08536, published February 2, 2026), which derives fine-grained rubrics from human-expert reports and scores agents on Information Recall, Analysis, and Presentation separately, rather than a single aggregate score ([arXiv abstract](https://arxiv.org/abs/2601.08536)). It evaluated OpenAI-GPT-o3 Deep Research, Gemini-3-Pro Deep Research, Gemini-2.5-Pro Deep Research, Perplexity Research, Grok Deep Search, and several Chinese-market agents as of a November 2025 snapshot — meaning it predates OpenAI's Feb-2026 GPT-5.2 rebuild and Google's Apr-2026 Gemini-3.1 rebuild, so results should be read as dated by roughly 3–6 months relative to current shipping versions. Its finding: no single agent dominates across all three axes. OpenAI-GPT-o3 and Gemini-3-Pro led Recall; OpenAI-GPT-o3 and Gemini-2.5-Pro led Analysis/synthesis quality; Grok Deep Search and Gemini-3-Pro led Presentation/readability; Perplexity Research consistently placed in the middle cluster on Recall and Analysis. This is the one comparison in this space that looks methodologically rigorous (academic peer-reviewable rubric construction) rather than marketing or SEO content. By contrast, numerous 2026 blog "comparisons" (e.g., a solo-developer benchmark claiming Perplexity Ultra and Claude Code tied at 97% accuracy, or aggregator sites citing OpenAI accuracy at 75.8–81.8%) come from low-transparency, non-peer-reviewed blog and SEO sources with unclear methodology; they are cited in resources.md but should be weighted far below DeepResearch Bench II. Separately, the general-purpose **BrowseComp** web-browsing leaderboard (benchlm.ai, accessed July 18, 2026) ranks underlying frontier LLMs (GPT-5.6 Sol leads at 92.2%) rather than the packaged deep-research products themselves, so it is only indirectly informative about which vendor's *product* wins.

## Trade-offs across the category

**Quality**: DeepResearch Bench II suggests OpenAI and Google's higher-end agents (o3/GPT-5.2-class, Gemini 3-Pro-class) generally out-recall and out-analyze Perplexity's and Grok's offerings, while Grok and Gemini edge out on report readability/presentation — but no vendor wins across the board, and the newest model generations (GPT-5.2+, Gemini 3.1) have not yet been independently re-benchmarked as of this writing. **Cost**: Perplexity Pro is $20/mo (20 research queries/day, 50 Labs queries/month) with Max at $200/mo adding a multi-model "Council" across Claude Opus, GPT-5.2, and Gemini 3 Pro ([Perplexity pricing roundup](https://suprmind.ai/hub/perplexity/pricing/) — cross-check with [PricePerToken](https://pricepertoken.com/subscriptions/perplexity)); OpenAI gates Deep Research by monthly query count rather than dollars (Free: 5/mo lightweight; Plus/Team/Enterprise: 25/mo, 15 lightweight; Pro $200/mo: 250/mo — figures reported as of mid-2025 and not confirmed refreshed post-GPT-5.2 rebuild, flagged as possibly stale); Google's new API is usage-priced per task (~$1–3 standard, ~$3–7 Max per Google's own docs, though a third-party cost analysis estimated Google Search grounding alone adds $1.12–2.24/task on top — these two source families disagree on exact bounds, flagged as a discrepancy) with a free consumer allowance of 5 Deep Research reports/month. **Transparency**: all three now show intermediate search queries and final source lists; Google additionally streams reasoning steps and offers a "review the plan before running" option; none expose true model chain-of-thought, and none publish their source-ranking/credibility-scoring logic. **Lock-in**: outputs are simple exportable documents (no proprietary format), but the underlying research process, model choice, and query allowances are entirely vendor-controlled — switching vendors means re-running research from scratch, and none offer a way to export or replay the agent's search trace elsewhere, unlike open-source agent frameworks where the orchestration code is portable. **Privacy**: Google and Perplexity increasingly connect these agents to private data via MCP/file-search/uploaded documents, and Anthropic's Research feature explicitly reads Gmail/Calendar/Drive content — all raise the same question of what gets retained/used for training, and vendor privacy policies (not fetched in this research pass) should be checked before feeding proprietary data to any of these.

## Getting started as a developer

To use OpenAI Deep Research: subscribe to ChatGPT Plus ($20/mo) or Pro ($200/mo) for the highest monthly allowance, select "Deep Research" in the composer, or call it programmatically via the Responses API with a deep-research-capable model. To use Gemini Deep Research: a Google AI Pro ($19.99/mo) or Ultra subscription unlocks unlimited runs in the Gemini app: for API-level automation, request access to the `deep-research-preview-04-2026` / `deep-research-max-preview-04-2026` agents in the Interactions API (public preview, pay-per-task, see [ai.google.dev docs](https://ai.google.dev/gemini-api/docs/interactions/deep-research)). To use Perplexity: a Pro subscription ($20/mo) gives 20 research queries/day and Labs access; the `sonar-deep-research` model is also directly callable via the Perplexity API (pay-per-token plus per-search fees) for programmatic use, documented with a full official cookbook of examples. As a practical baseline for benchmarking a homegrown research agent: run the same prompt set through ChatGPT Plus Deep Research, Gemini Deep Research (Pro tier), and Perplexity Pro Deep Research side by side, since all three are cheaply accessible at the $20/mo tier; score outputs against a rubric like DeepResearch Bench II's (recall / analysis / presentation, separately) rather than a single pass/fail, since no single vendor wins across all three dimensions; and treat any vendor-quoted benchmark number (HLE, "win rate," etc.) as marketing until corroborated by an independent, dated source.
