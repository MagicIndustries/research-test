# Annotated Resources — Open-Source Research Frameworks

> AS_OF: 2026-07-18. Every entry was surfaced by this project's research pipeline (registry numbers in brackets). Accessibility: all public. Stars/releases read directly from GitHub 2026-07.

## Repositories (the frameworks themselves)

- **[22] assafelovic/gpt-researcher** — https://github.com/assafelovic/gpt-researcher — Apache-2.0, 28.4k stars, v3.5.1 (2026-06-23). The pragmatic default: planner/executors/publisher, deep (tree) research mode, MCP support, explicit ~$0.40/run cost claim. Best repo to copy *cost discipline* from.
- **[23] langchain-ai/open_deep_research** — https://github.com/langchain-ai/open_deep_research — MIT, ~12k stars. Supervisor/researcher LangGraph reference; per-stage model config; RACE 0.4943 (w/ GPT-5) on Deep Research Bench, #6 (self-reported). Best repo to copy *architecture* from.
- **[25] bytedance/deer-flow** — https://github.com/bytedance/deer-flow — MIT, 77.3k stars, v2.0.0 (2026-06-25). Lead agent + parallel sub-agents + sandboxes + persistence; the "super agent harness" endgame of the category. Heaviest to operate.
- **[26] stanford-oval/storm** — https://github.com/stanford-oval/storm — MIT, 30.1k stars, but v1.1.0 (2025-01) is stale. Peer-reviewed (NAACL/EMNLP 2024) perspective-guided outline-first writing; Co-STORM adds human-in-the-loop discourse. Best repo to copy *question-generation and outlining* from.
- **[27] huggingface/smolagents — open_deep_research example** — https://github.com/huggingface/smolagents/blob/main/examples/open_deep_research/README.md — CodeAgent replication of OpenAI DR; 55% GAIA pass@1 vs 67% original (self-reported). The honest yardstick for the open-vs-closed gap.
- **[28] jina-ai/node-DeepResearch** — https://github.com/jina-ai/node-DeepResearch — TypeScript iterative search-read-reason loop with token budgets; minimal architecture worth reading in an afternoon.
- **[29] LearningCircuit/local-deep-research** — https://github.com/LearningCircuit/local-deep-research — community; fully local/encrypted, 20+ strategies, 10+ search engines, ~95% SimpleQA (self-reported, unaudited).
- **[30] khoj-ai/khoj** — https://github.com/khoj-ai/khoj — self-hostable "AI second brain" with deep-research mode; broader than research alone.
- **[33] btahir/open-deep-research** — https://github.com/btahir/open-deep-research — community; lightweight Firecrawl-based report generator; good minimal starting point.
