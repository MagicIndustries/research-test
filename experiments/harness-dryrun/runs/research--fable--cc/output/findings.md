# Current Node.js LTS release line and end-of-life date

As of July 2026, the current Active LTS release line is **Node.js v24 "Krypton"**. Per the [official Node.js release schedule](https://github.com/nodejs/Release) ([schedule.json](https://raw.githubusercontent.com/nodejs/Release/main/schedule.json)), v24 entered Active LTS on 2025-10-28, transitions to Maintenance LTS on 2026-10-20, and reaches its scheduled **end-of-life on 2028-04-30**.

One other line is still in LTS: **Node.js v22 "Jod"**, now in Maintenance LTS, with end-of-life scheduled for 2027-04-30. The [Node.js previous-releases page](https://nodejs.org/en/about/previous-releases) confirms v24 (Krypton) as Active LTS and v22 (Jod) as Maintenance LTS, with v26 as the Current (non-LTS) release.
