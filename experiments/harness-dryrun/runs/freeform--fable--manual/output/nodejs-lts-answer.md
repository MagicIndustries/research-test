# Node.js LTS — quick answer

The current Node.js LTS release line is **Node.js 24**, which entered Active LTS in late October 2025. Under the Node.js release policy, each LTS line gets roughly 12 months of Active LTS followed by maintenance, ending 30 months after the line first went LTS.

That puts Node.js 24's scheduled end-of-life in **April 2028**, per the Node.js release schedule (https://github.com/nodejs/Release). Note: written from working knowledge without re-fetching the schedule, so dates should be checked against the source before relying on them.
